position: static;

Position Static digunakan untuk mengatur elemnt menjadi statis secara default

element ini diatur menggunakan posisi: static




position: relative;

Position relative digunakan untuk mengatur elemnt akan terletak pada posisi normal

element ini diatur menggunakan posisi: relative




position: fixed;

Position fixed digunakan untuk mengatur elemnt menjadi sifat tetap

element ini diatur menggunakan posisi: fixed




position: absolute;

Position absolute digunakan untuk mengatur elemnt menjadi posisi relatif dengan elemtn lain yang mendahuluinya yang terdekat

element ini diatur menggunakan posisi: absolute




position: sticky;

Silahkan di Scroll agar tau fungsi sticky bekerja

Nama aku Sticky !
